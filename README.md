# # Active Directory Home Lab

## Overview

This project demonstrates the deployment and configuration of Active Directory Domain Services (AD DS) in a Windows Server 2022 virtual machine. The lab includes installing Active Directory, creating a new domain, managing users, and assigning group memberships.

## Objectives

* Install Windows Server 2022
* Deploy Active Directory Domain Services (AD DS)
* Create a new domain forest (lab.local)
* Create and manage user accounts
* Create and manage security groups
* Verify group memberships using Active Directory Users and Computers (ADUC)

## Environment

* VirtualBox
* Windows Server 2022 Standard Evaluation
* Active Directory Domain Services (AD DS)

## Tasks Completed

### 1. Windows Server Installation

* Installed Windows Server 2022 in a virtual machine
* Selected Windows Server 2022 Standard Evaluation (Desktop Experience)
* Configured virtual disk storage

### 2. Active Directory Installation

* Added the Active Directory Domain Services role
* Installed required management tools
* Verified successful installation

### 3. Domain Configuration

* Created a new Active Directory forest
* Configured the root domain:

  * lab.local

### 4. User Management

Created the following domain user accounts:

* IT Admin
* Help Desk
* Test User

### 5. Group Management

Created and verified security group memberships:

* ITAdmins
* HelpDesk

### 6. Verification

Verified:

* Successful Active Directory deployment
* User account creation
* Security group creation
* Proper group membership assignments

## Skills Demonstrated

* Windows Server Administration
* Active Directory Domain Services
* Domain Controller Deployment
* User Account Management
* Security Group Administration
* Virtualization with VirtualBox
* Identity and Access Management (IAM)
